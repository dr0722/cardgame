# 海洋動物閃卡遊戲圖片

這個資料夾用來儲存遊戲所需的海洋動物圖片。

## 需要的圖片

請添加以下動物的圖片到此資料夾：

1. penguin.png - 企鵝圖片
2. dolphin.png - 海豚圖片
3. shark.png - 鯊魚圖片
4. whale.png - 鯨魚圖片

## 圖片來源建議

可以使用免費的兒童友善圖片，來源如：
- Pixabay
- Unsplash
- Freepik (注意使用免費授權的圖片)
- Flaticon (適合卡通風格的圖示)

## 圖片規格

為了最佳顯示效果，建議:
- 圖片尺寸至少為 400x400 像素
- 使用 PNG 格式以支援透明背景
- 選擇卡通風格、色彩鮮豔的圖片，適合兒童使用
